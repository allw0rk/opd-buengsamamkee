วิสัยทัศน์ · Vision

"มีมาตรฐาน ปลอดภัย ใส่ใจทุกคน เพื่อสุขภาพดีของประชาชน"

พันธกิจ · Mission
  • 1ดูแลผู้ป่วยแบบองค์รวม ครอบคลุมการส่งเสริม ป้องกัน รักษา และฟื้นฟู
  • 2เชื่อมต่อการดูแลสู่เครือข่ายชุมชน เพื่อสุขภาพที่ต่อเนื่องและยั่งยืน
  • 3พัฒนาทีมงานให้มีสมรรถนะตามมาตรฐานวิชาชีพ และมีความสุขในการทำงาน
  • 4บริหารทรัพยากรอย่างมีประสิทธิภาพ คุ้มค่า และโปร่งใส
ค่านิยม · Core Values (รวย)
Core values
ค่านิยม รวย
HAI core values
รับผิดชอบ
R = Responsibility
Accountability → ความรับผิดชอบต่อผลลัพธ์ของผู้ป่วยและงานที่ทำ
Focus on Patient → คำนึงถึงความปลอดภัยและประโยชน์ของผู้ป่วยเป็นหลัก
มีวินัย
D = Discipline
Continuous Improvement → ปฏิบัติตามมาตรฐานอย่างสม่ำเสมอ
Systems Thinking → ทำงานตามระบบ ไม่ทำตามความเคยชิน
ยึดมั่นบริการ
S = Service Commitment
Patient Focus / Customer Focus → ใส่ใจผู้ป่วยเป็นศูนย์กลาง
Teamwork → ทำงานร่วมกันเพื่อให้บริการที่ดีที่สุด
